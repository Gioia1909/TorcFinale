Detailed Description

ManualDriver consente il controllo manuale dell'auto tramite tastiera e la registrazione dei dati su file CSV per addestramento supervisato. È possibile alternare tra modalità automatica e manuale, registrare i comandi, e salvare i sensori in tempo reale ogni 150ms.
