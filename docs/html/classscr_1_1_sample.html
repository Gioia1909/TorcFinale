Detailed Description

Rappresenta un singolo campione del dataset, contenente input sensoriali e output delle azioni del veicolo.

scr.Sample.Sample ( String line)
inline

Costruttore che crea un Sample a partire da una riga CSV.

Assume che le ultime 4 colonne siano i valori target: accelerazione, frenata, sterzata e marcia.

Parameters
lineStringa CSV contenente feature e target separati da virgole.

< Feature: tutte le colonne tranne le ultime 4

< Target: ultime 4 colonne

< Accelerazione

< Frenata

< Sterzata

< Marcia
